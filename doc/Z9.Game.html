Properties
Name Type Attributes Default Description
canvasId string HTML ID of canvas element to create Game on.
background string <optional>
 Fill color (if ommitted clear is used instead).
simpleLoop boolean <optional>
 false When set to true, a simple main loop will be used. This is an alternative version of the core game loop that only does drawing, and leaves updates to the programmer. Use this if your game does not require timed events and relies on user events.
fps number <optional>
 60 Game FPS (void when simpleLoop=true).
framerate number <optional>
 60 Game frame rate (void when simpleLoop=true).
gameStates array <optional>
 List of states to be initialized upon game run. If states are left uninitialized, they will initialize once as they activate for the first time. You can safely omit this argument if your states can load independently. Use initStates() to initialize states manually.
